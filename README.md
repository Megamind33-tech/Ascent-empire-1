# Ascent Realms

Multi-nation Babylon.js + Rapier political city-builder starter.

## Features
- 10 nations with distinct capital generation
- Coastal nations get seaports and moving ships
- Inland nations get airstrips
- Building placement pads to avoid clipping into roads
- Day/night cycle, fog, mobile-first density
- Save/load plus automatic checkpoints
- Opening 2D HTML setup UI for player name, nation, and starting position

## Run
```bash
npm install
npm run dev
```
